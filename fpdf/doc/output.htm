Output

Output([string file])

Description

Sauve le document PDF dans un fichier ou l'envoie au navigateur. Dans ce dernier cas, les en-têtes HTTP sont d'abord émis.
La méthode commence par appeler Close() si nécessaire pour terminer le document.

Paramètres

file
Le nom du fichier. S'il est vide ou non renseigné, le document est envoyé au navigateur.

Voir

Close().
