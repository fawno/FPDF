MultiCell

MultiCell(double w, double h, string txt [, int border [, string align [, int fill]]])

Description

This method allows printing text with line breaks. They can be automatic (as soon as the text reaches the right border of the cell) or explicit (via the \n character). As many cells as necessary are output, one below the other.
Text may be aligned, centered or justified. The cell block may be framed and the background painted.

Parameters

w
Width of cells. If 0, they extend up to the right margin of the page (the right margin equals the left one).
h
Height of cells.
txt
String to print.
border
Indicates if a border must be drawn around the cell block (0: no, 1: yes). Default value: 0.
align
Sets the text alignment. Possible values are:
  • L: left alignment
  • C: center
  • R: right alignment
  • J: justification (default value)
fill
Indicates if the cell background must be painted (1) or transparent (0). Default value: 0.
